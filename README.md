# Breast Cancer Classification

Breast cancer classification project using machine learning models and feature engineering.

## Features

* Breast cancer data analysis and visualization
* Data preprocessing and standardization
* Classification using decision tree, random forest, KNN, logistic regression, naive Bayes, and XGBoost
* Cross-validation and recall-based model comparison
* Confusion matrix visualization
* Feature engineering for improved model performance

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* XGBoost

## How to run

jupyter notebook
